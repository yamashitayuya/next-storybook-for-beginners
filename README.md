# Next storybook for Beginners

Next.js + Storybook + Tailwind CSS + TypeScript

## Prerequisites

- [Node.js](https://nodejs.org/) v20.11.1 or higher
- [pnpm](https://pnpm.io/ja/installation) v9.2.0 or higher

### Project setup

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

mock serverを起動する

```bash
pnpm mock
```

APIドキュメントを表示する

```bash
pnpm document
```

### Storybook

```bash
pnpm storybook
```

### Build

```bash
pnpm build
```
